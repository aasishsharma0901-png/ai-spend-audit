# FAQ

## Is this actually free?

Yes.

There is:

* no login
* no credit card
* no paywall

You receive the full audit instantly.

Email capture happens only after results are shown and is optional.

---

## How accurate is the pricing data?

All pricing information is sourced from official vendor pricing pages and verified during the final submission week.

Pricing references are documented publicly inside:

```txt id="jlwmg2"
PRICING_DATA.md
```

Enterprise pricing is treated separately because vendor contracts are usually negotiated individually.

---

## Will I get spammed if I enter my email?

No.

You receive:

* a copy of your audit
* optional follow-up only if significant savings opportunities are detected

No automated marketing sequences were implemented for this MVP.

---

## What is Credex?

Credex helps companies access discounted AI infrastructure credits sourced from:

* overcommitted contracts
* unused vendor allocations
* infrastructure resale opportunities

For companies with meaningful AI spend, this can reduce operational costs without changing workflows.

---

## Which tools are currently supported?

Current support includes:

* Cursor
* GitHub Copilot
* Claude
* ChatGPT
* Anthropic API
* OpenAI API
* Gemini
* Windsurf

Additional tools can be added as pricing demand and user requests evolve.
